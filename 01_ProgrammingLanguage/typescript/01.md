
# TypeScript 기본

## TypeScript란?

JavaScript + 타입 시스템

타입 체크를 통해 대규모 프로젝트에서 오류를 줄인다.

React 실무에서 필수.

---

# 1-1. 기본 타입

```typescript
let name: string = "Junhee";
let age: number = 25;
let isStudent: boolean = true;
```

---

# 1-2. 배열 타입

```typescript
let nums: number[] = [1, 2, 3];
```

---

# 1-3. 함수 타입

```typescript
const add = (a: number, b: number): number => {
  return a + b;
};
```

---

# 1-4. Interface

객체 구조 정의.

```typescript
interface User {
  name: string;
  age: number;
}
```

사용:

```typescript
const user: User = {
  name: "Kim",
  age: 20
};
```

React에서 엄청 중요. javascript 보다 더 엄격하게 타입을 체크하기 위함.

---

# 1-5. Optional

```typescript
interface User {
  name: string;
  age?: number;
}
```

`?`는 선택값. 즉, nullable

---

# 1-6. Generic

재사용 가능한 타입.

```typescript
function getValue<T>(value: T): T {
  return value;
}
```

사용:

```typescript
getValue<string>("hello");
getValue<number>(123);
```

React Query, Axios 등에서 계속 보인다.
