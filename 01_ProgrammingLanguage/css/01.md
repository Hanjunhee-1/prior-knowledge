# CSS 기본

프론트 UI 만드는 데 필요.

---

# 1-1. Flexbox

가장 중요하다.

```css
.container {
  display: flex;
  justify-content: center;
  align-items: center;
}
```

## 핵심 속성

| 속성              | 의미    |
| --------------- | ----- |
| justify-content | 가로 정렬 |
| align-items     | 세로 정렬 |
| gap             | 요소 간격 |

---

# 1-2. Grid

2차원 레이아웃.

```css
.container {
  display: grid;
  grid-template-columns: 1fr 1fr;
}
```

---

# 1-3. 반응형

모바일 대응.

```css
@media (max-width: 768px) {
  .container {
    flex-direction: column;
  }
}
```

---

# CSS 공부 목표

* 카드 UI 제작
* 반응형 레이아웃
* Flexbox 자유롭게 사용
