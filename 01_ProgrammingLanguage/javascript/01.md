# 1. JavaScript (ES6+) 기본

## JavaScript가 뭔가?

웹 브라우저에서 동작하는 프로그래밍 언어다.
현재는 프론트엔드뿐 아니라 서버(Node.js), AI 툴링, 자동화까지 전부 사용된다.

React도 JavaScript 기반이다.

---

# 1-1. 변수 선언: let / const

```javascript
let age = 20;
age = 21;

const name = "Junhee";
// name = "Kim"; // 오류
```

## 차이

| 키워드   | 재할당 |
| ----- | --- |
| let   | 가능  |
| const | 불가능 |

실무에서는 기본적으로 `const`를 쓰고, 값이 바뀌면 `let` 사용.

# 1-3. 함수

## 일반 함수

```javascript
function add(a, b) {
  return a + b;
}
```

## 화살표 함수

```javascript
const add = (a, b) => {
  return a + b;
};
```

짧게 쓰면:

```javascript
const add = (a, b) => a + b;
```

React에서 매우 많이 사용한다.

---

# 1-4. 배열 다루기

## map

배열 변환.

```javascript
const nums = [1, 2, 3];

const result = nums.map(n => n * 2);

console.log(result);
// [2, 4, 6]
```

---

## filter

조건 필터링.

```javascript
const nums = [1, 2, 3, 4];

const even = nums.filter(n => n % 2 === 0);

console.log(even);
// [2, 4]
```

---

## find

```javascript
const users = [
  { id: 1, name: "Kim" },
  { id: 2, name: "Lee" }
];

const user = users.find(u => u.id === 2);
```

# 1-6. Spread 문법 (...)

```javascript
const arr1 = [1, 2];
const arr2 = [...arr1, 3];

console.log(arr2);
// [1, 2, 3]
```

객체 복사:

```javascript
const user = {
  name: "Kim"
};

const newUser = {
  ...user,
  age: 20
};
```

# 1-7. 비동기(async/await)

API 호출, DB 통신, AI 요청 모두 비동기다.

## Promise

```javascript
const fetchData = () => {
  return new Promise(resolve => {
    setTimeout(() => {
      resolve("완료");
    }, 1000);
  });
};
```

---

## async/await

```javascript
const getData = async () => {
  const result = await fetchData();

  console.log(result);
};
```

## 핵심

| 키워드   | 의미            |
| ----- | ------------- |
| async | 비동기 함수        |
| await | 작업 끝날 때까지 기다림 |

---

# 1-8. fetch API

서버 요청 기본.

```javascript
const getUsers = async () => {
  const response = await fetch("https://api.example.com/users");

  const data = await response.json();

  console.log(data);
};
```
